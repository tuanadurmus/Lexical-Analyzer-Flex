# Lexical Analyzer (Flex)

This repository contains my **Flex (.flx) lexer implementation**.

The lexer scans an input stream and produces tokens according to the rules defined in the `.flx` file.

---

## My Contribution

- Implemented the lexer specification in **`lexical-analyzer.flx`**
- Defined token patterns using regular expressions (Flex rules)
- Handled token emission and any required lexical edge cases (e.g., whitespace, invalid tokens, etc.)
